# Jguida941 Tap-setup-proof

## How do I install these formulae?

`brew install jguida941/tap-setup-proof/<formula>`

Or `brew tap jguida941/tap-setup-proof` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "jguida941/tap-setup-proof"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
